## Reporting a Vulnerability

Please report security vulnerabilities by creating a new security advisory on GitHub or by contacting the maintainers directly. Do not open public issues for security vulnerabilities.

We will try to review your report as soon as possible and provide feedback on the expected resolution timeline.
